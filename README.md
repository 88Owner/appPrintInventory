# appPrintInv

## Cài đặt

```powershell
cd c:\Users\ADMIN\appPrintInv
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```


## Chạy app

```powershell
python -m app
```

## Build file EXE (có icon)

```powershell
cd c:\Users\ADMIN\appPrintInv
.\build_exe.ps1
```

File tạo ra ở `dist\appPrintInv.exe`.
